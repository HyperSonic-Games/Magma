## How To Build From Source

### Dependencies

- SDL2_image-2.8.2 (For Textures)
- SDL2_ttf-2.22.0 (For Fonts and other text stuff)
- SDL2-2.28.5 (For rendering)

### Windows

To build on Windows, place the Dependency Folders in C://Libs and set the compile target to X64.

# FAQ

## What Platforms does Magma support?

Magma supports any 64-bit system where SDL2 can run, but there are no guarantees that everything will work because it was primarily developed with Windows in mind.
