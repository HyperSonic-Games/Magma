OpenTCPSocket

OpenTCPSocket (
    port: u16,
    addr_type: IPAddrFamily,
    addr: IPAddr,
    is_blocking: bool = false,
    send_keepalive_msgs: bool = false
) -> Maybe

OpenTCPSocket Creates and configures a TCP socket optimized for low-latency game networking.

Parameters:
  • port: The local port to bind the socket to. If 0, no bind occurs (useful for clients).
  • addr_type: The IP address family for the socket (e.g., IPv4, IPv6).
  • addr: The local IP address to bind to. Used when acting as a server.
  • is_blocking: If true, socket operations block until completion. If false, the socket is non-blocking and suitable for integration into a game engine's main loop.
  • send_keepalive_msgs: If true, the OS will periodically send TCP keep-alive packets to detect dead peers. Typically false for real-time games; use custom heartbeat/ping logic instead.
Returns: A Maybe(TCP_SocketHandle)

SendTCPSocket

SendTCPSocket (socket: TCP_SocketHandle, data: []byte)

SendTCPSocket Sends a buffer of bytes over a TCP socket.

Parameters:
  • socket: The TCP socket handle to send data through.
  • data: A slice of bytes containing the data to send.
Returns: The number of bytes successfully written. and true if the send completed without error; false if an error occurred.

ReciveTCPSocket

ReciveTCPSocket (socket: TCP_SocketHandle, buf: []byte)

ReciveTCPSocket Receives data from a TCP socket into the provided buffer.

Parameters:
  • socket: The TCP socket handle to read data from.
  • buf: A slice of bytes that will be filled with received data.
Returns: The number of bytes successfully read and true if success; false if error

CloseTCPSocket

CloseTCPSocket (socket: TCP_SocketHandle)

CloseTCPSocket Closes a open TCP socket

Parameters:
  • socket: The handle to an open TCP socket
