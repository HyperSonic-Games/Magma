OpenUDPSocket

OpenUDPSocket (
    port: u16,
    addr_type: IPAddrFamily,
    addr: IPAddr,
    is_blocking: bool = false,
    allow_broadcast: bool = false
) -> Maybe

OpenUDPSocket Creates and configures a UDP socket optimized for low-latency game networking.

Parameters:
  • port: The local port to bind the socket to. If 0, no bind occurs (useful for clients).
  • addr_type: The IP address family for the socket (e.g., IPv4, IPv6).
  • addr: The local IP address to bind to. Used when acting as a server or listener.
  • is_blocking: If true, socket operations block until completion. If false, the socket is non-blocking and suitable for integration into a game engine's main loop.
  • allow_broadcast: If true, enables sending to and receiving from broadcast addresses.
Returns: A Maybe(UDP_SocketHandle)
