InitTCPServerIPV4

InitTCPServerIPV4 (
    addr: m_net.IPV4,
    port: u16,
    msg_handler: ServerMsgHandler,
    mutex_ptr: ^sync.Mutex = nil,
)

InitTCPServerIPV4 Creates a new server using the IPv4 protocol

Parameters:
  • addr: The IPv4 address the server will bind to
  • port: The TCP port number to listen on
  • msg_handler: A user-defined function that handles incoming packets
  • mutex_ptr: Optional pointer to a user-provided mutex to safely access shared data
Returns: (^TCPServer, bool) Returns a pointer to the newly created server and true on success, false if socket creation failed
