InitUDPServerIPV4

InitUDPServerIPV4 (
    addr: m_net.IPV4,
    port: u16,
    msg_handler: ServerMsgHandler,
    mutex_ptr: ^sync.Mutex = nil,
)

InitUDPServerIPV4 Creates a new UDP server using IPv4

Parameters:
  • addr: The IPv4 address to bind the server
  • port: The UDP port number
  • msg_handler: User-defined packet handler
  • mutex_ptr: Optional pointer to a mutex for thread safety
Returns: (^UDPServer, bool) Returns a pointer to the server and true on success

RunUDPServer

RunUDPServer (server: ^UDPServer)

UDPServerTick Processes incoming UDP packets and sends responses / @private UDPServerTick :: proc(server: ^UDPServer) { buf := make([]byte, 4096) // Temporary buffer for incoming packet client_addr := Basic.IPAddr{} // To store the client's address // Receive a packet from any client (non-blocking) bytes_read, address, port := Basic.ReciveUDPSocket(server.socket, buf) if bytes_read == 0 { return // No packet received } packet := buf[:bytes_read] // Protect the user handler with mutex if provided if server.data_mutex != nil { sync.mutex_lock(server.data_mutex) } response := server.msg_handler(packet, client_addr) if server.data_mutex != nil { sync.mutex_unlock(server.data_mutex) } // Send a response if the handler returned data if len(response) > 0 { _, _ = Basic.SendUDPSocket(server.socket, response, client_addr, port) } } / Worker loop for the server running in a separate thread / @private Worker :: proc(thread_cxt: ^thread.Thread) { server := cast(^UDPServer)thread_cxt.data for sync.atomic_load(server.internal_running) { UDPServerTick(server) thread.yield() // Allow other threads to run } } / RunUDPServer Starts the server in a new worker thread

Parameters:
  • server: Pointer to the UDPServer instance
  • thread_cxt: Pointer to the thread context containing the server
