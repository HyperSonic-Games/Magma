ResetWindowFlags

ResetWindowFlags (state: ^WindowState)

ResetWindowFlags sets all the window state flags to a known state (should be called after game logic in the main loop)

Parameters:
  • state: a pointer to a stucture that holds the window state data

HandleEvents

HandleEvents (mouse: ^Mouse, keyboard: ^Keyboard, win: ^WindowState)

HandleEvents When called updates all of the event structures passed to it

Parameters:
  • mouse: a pointer to a stucture that holds mouse data
  • keyboard: a pointer to a structure that contains all ot the keys and mod keys being pressed
  • win: a pointer to a structure that holds general window state (fullscreen, quit state, ect)
