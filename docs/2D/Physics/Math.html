SetMeterToPixelScale

SetMeterToPixelScale (new_scale: f32)

SetMeterToPixelScale sets a new meters-to-pixels scale value at runtime. @note Panics if new_scale is <= 0

Parameters:
  • new_scale: the new scale factor to use

GetMeterToPixelScale

GetMeterToPixelScale () -> f32

GetMeterToPixelScale returns the current meters-to-pixels scale factor.

Returns: f32 current scale factor

MetersToPixels

MetersToPixels (meters: f32) -> f32

MetersToPixels converts a distance in meters to pixels using the current scale factor.

Parameters:
  • meters: distance in meters
Returns: f32 equivalent distance in pixels

PixelsToMeters

PixelsToMeters (pixels: f32) -> f32

PixelsToMeters converts a distance in pixels to meters using the current scale factor.

Parameters:
  • pixels: distance in pixels
Returns: f32 equivalent distance in meters
