CreateDistanceJoint

CreateDistanceJoint (
    world: World,
    obj_a: Obj,
    obj_b: Obj,
    anchor_a: Types.Vector2f,
    anchor_b: Types.Vector2f,
    rest_length: Types.Meters,
    min_length: Types.Meters,
    max_length: Types.Meters,
    handle_collision: bool = true
) -> JointHandle

CreateDistanceJoint creates a fixed-length joint between two objects. The distance between anchor points on the two bodies is constrained.

Parameters:
  • world: Physics world
  • obj_a: First object
  • obj_b: Second object
  • anchor_a: Local anchor point on obj_a
  • anchor_b: Local anchor point on obj_b
  • rest_length: Target length of the joint
  • min_length: Minimum allowed length
  • max_length: Maximum allowed length
  • handle_collision: Whether connected bodies should collide
Returns: JointHandle Handle to the created joint
