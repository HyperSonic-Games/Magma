CreateDistanceJoint

CreateDistanceJoint (
    world: World,
    obj_a: Obj,
    obj_b: Obj,
    anchor_a: Types.Vector2f,
    anchor_b: Types.Vector2f,
    rest_length: Meters,
    min_length: Meters,
    max_length: Meters,
    handle_collision: bool = true
) -> JointHandle

CreateDistanceJoint creates a distance joint between two objects.

Parameters:
  • world: the physics world
  • obj_a: first object
  • obj_b: second object
  • anchor_a: local anchor point on obj_a
  • anchor_b: local anchor point on obj_b
  • rest_length: the target distance between the anchors
  • min_length: minimum allowed distance
  • max_length: maximum allowed distance
  • handle_collision: whether the connected bodies should collide
Returns: JointHandle handle to the created joint
