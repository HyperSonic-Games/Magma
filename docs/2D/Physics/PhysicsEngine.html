InitPhysicsEngine

InitPhysicsEngine (grav_vector: Types.Vector2f, can_sleep := true, collision_is_continuous := true) -> World

InitPhysicsEngine initializes a Box2D world with gravity and settings.

Parameters:
  • grav_vector: gravity vector (x, y)
  • can_sleep: whether bodies can go to sleep for performance
  • collision_is_continuous: enables continuous collision detection
Returns: World physics world handle

DestroyPhysicsEngine

DestroyPhysicsEngine (world: World)

DestroyPhysicsEngine destroys a physics world and frees resources.

Parameters:
  • world: the world to destroy

CreateStaticBody

CreateStaticBody (world: World,
    pos: Types.Vector2f,
    angle: Types.Radians = 0,
    can_sleep := true,
    no_physics_rot := true
)-> Obj

CreateStaticBody creates a static (immovable) body in the physics world.

Parameters:
  • world: the physics world
  • pos: body position
  • angle: body angle in radians
  • can_sleep: whether the body can sleep
  • no_physics_rot: prevents physics-based rotation if true
Returns: Obj handle to the body

AddPhysicsPolygonToBody

AddPhysicsPolygonToBody (
    obj: Obj, 
    vertices: [][2]f32, 
    radius: f32,
    density: f32,
    mat_registry_name: string
) -> ObjPhysicsHandle

AddPhysicsPolygonToBody attaches a polygon shape with physics properties to a body.

Parameters:
  • obj: the body to attach to
  • vertices: array of polygon vertices
  • radius: rounding radius for edges
  • density: material density
  • mat_registry_name: key to look up material in registry
Returns: ObjPhysicsHandle handle to the shape

  • delta_time: delta time multiplier
